## How to Use These Files

### Quick Start (Manual Setup)

```bash
# 1. Create project directory
mkdir clawless && cd clawless

# 2. Download all files above into this directory

# 3. Initialize
npm install

# 4. Configure
cp .env.example .env
# Edit .env with your API keys

# 5. Start
npm run quickstart
```

### With Cursor AI

```bash
# 1. Create folder with all files
mkdir clawless && cd clawless
# Download all files here

# 2. Open in Cursor
cursor .

# 3. Tell Cursor:
"Read HANDOFF.md and implement Phase 0 from IMPLEMENTATION_PLAN.md"
```

### With Aider

```bash
cd clawless  # folder with all downloaded files

aider --read HANDOFF.md --read IMPLEMENTATION_PLAN.md --read ARCHITECTURE.md

# Then type:
"Implement Phase 0: Foundation as described in IMPLEMENTATION_PLAN.md"
```

### tsconfig.json
TypeScript configuration:
- Strict mode enabled
- Path aliases (@/*)
- Next.js integration
- Target ES2022

### .env.example
Template for environment variables:
- ANTHROPIC_API_KEY (required)
- GITHUB_TOKEN (required)
- SLACK_TOKEN (optional)
- Ollama configuration
- Webhook secrets

### .gitignore
Git ignore patterns:
- node_modules
- .env (secrets)
- data/state.db (user data)
- workflows/ (user workflows)
- dist/ build/

### examples-pr-reviewer.json
Example workflow that:
1. Extracts PR info (Ollama - free)
2. Fetches diff (no LLM needed)
3. Classifies risk level (Ollama ‚Üí Claude fallback)
4. Does security review (Claude - complex)
5. Posts comment with findings
6. Notifies on Slack

Shows the Intelligence Router in action!
